# Alpine Ascents User Guide

Welcome to the **Alpine Ascents** mountaineering portal. This guide will help you navigate the features of our platform.

## 1. Getting Started
To view the website, simply open the `index.html` file in any modern web browser.

## 2. Navigating the Portal
Use the **Header Navigation Menu** to quickly jump to different sections:
- **Home**: View the hero slideshow and introductory highlights.
- **About**: Learn about the history and types of mountaineering.
- **Details**: Explore technical details like techniques, sheltering, and hazards.
- **Records**: View world-record achievements in mountaineering.
- **Local Activities**: Find climbing clubs and organizations near you.
- **Stories**: Read success stories from past expeditions.
- **Gallery**: Browse high-quality images of mountain adventures.
- **Developments**: Stay updated on the latest climbing tech and gear.

## 3. Finding Local Activities
The "Local Activities" section uses **Geolocation Services**:
1. When prompted, click **"Enable Location"**.
2. Grant permission in your browser's security popup.
3. The site will automatically detect your city and display the nearest mountaineering clubs.
4. If location is denied, the system defaults to showing major global organizations.

## 4. Interactive Features
- **Status Box**: Click the info icon at the bottom right to view live statistics, including your local time and current safety metrics.
- **Slideshow**: Use the arrows or navigation dots on the home section to browse featured adventures.
- **Maps**: Click "View on Google Maps" on any club card to get directions or more info.

## 5. Safety & Responsibility
We recommend all users thoroughly read the **General Guidelines** section before planning any expedition. Always prioritize physical preparation and environmental responsibility (Leave No Trace).
