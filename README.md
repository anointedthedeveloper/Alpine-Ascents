## 🏔️ What is Alpine Ascents?

**Alpine Ascents** is a fully responsive, single-page mountaineering website — an informational and aspirational hub for climbers of all levels, from first-time trekkers to seasoned summiteers. Built entirely with **vanilla HTML, CSS, and JavaScript**. No frameworks, no fuss.

> _"The mountains are calling and I must go."_ — John Muir

## 🚀 Getting Started

No build tools. No installs. Just **open and climb**.

```bash
# 1. Clone the repo
git clone https://github.com/mmesomaonyeka301-code/alpine-ascents.git

# 2. Enter the project
cd alpine-ascents

# 3. Launch in browser
open index.html
```

> 💡 **Tip for the map:** Leaflet renders best over a local server. Use **VS Code Live Server** or:
> ```bash
> python3 -m http.server 8000
> # → Visit http://localhost:8000
> ```
